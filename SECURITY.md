# Security Policy

SparseSpline-FFN is research software and currently does not accept untrusted
model files, executable plugins, or network inputs at runtime.  Security issues
should still be handled privately.

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | yes |

## Reporting

Open a private security advisory on GitHub or contact the repository owner
directly.  Do not publish proof-of-concept exploits before a fix is available.
